# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities by emailing [nkishore@megam.io](mailto:nkishore@megam.io) instead of using the public issue tracker.

Include a description, reproduction steps, and potential impact.

We will acknowledge your report within 24 hours and provide updates every 48 hours during remediation.

## Vulnerability Disclosure

Significant vulnerabilities will be announced via GitHub Security Advisories and the [RustSec advisory database](https://github.com/RustSec/advisory-db) (tracked by `cargo audit`).
